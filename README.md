# Doubao 豆包 AI 图像生成工具

基于字节跳动火山引擎方舟平台 Doubao Seedream 模型的文生图/图生图 Web 工具。

提供可视化网页界面，支持输入提示词生成，支持上传图片进行编辑，自动保存所有历史图片，完整日志记录。

## 功能特性

- ✅ **文生图** - 输入提示词生成图片
- ✅ **图生图/图像编辑** - 上传原图，描述需要修改的内容
- ✅ **网页界面** - 可视化操作，方便使用
- ✅ **历史记录** - 网格展示所有历史生成图片，支持放大预览和下载
- ✅ **自动保存** - 所有图片保存到 `images/` 目录，按时间戳命名，不覆盖
- ✅ **完整日志** - 生成信息记录到 `generation.log`
- ✅ **耗时统计** - 显示每次生成耗时
- ✅ **安全** - API Key 保存在 `.env`，不会被 Git 追踪

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key

在 `.env` 文件中配置你的 API Key：
```bash
ARK_API_KEY=your_api_key_here
```

`.env` 默认在 `.gitignore` 中，不会提交到 Git，保证密钥安全。

### 3. 启动服务

```bash
python app.py
```

### 4. 打开浏览器

访问 http://localhost:5000 即可使用。

## 使用说明

### 文生图

1. 在文本框输入你的提示词
2. 点击"开始生成"
3. 等待生成完成，结果会显示在页面
4. 历史图片会自动展示在下方网格中

### 图生图（图像编辑）

1. 点击"选择文件"上传需要编辑的原图
2. 在文本框描述你需要修改的内容（例如："将蓝色发带变成粉色，去掉胡子，增加一只鹦鹉"）
3. 点击"开始生成"
4. 生成完成后会显示编辑结果

### 查看历史

页面下方展示所有生成过的图片：
- 点击图片可以放大查看
- 点击"下载"下载图片到本地
- 按修改时间倒序排列，最新的在前

## 命令行版本（可选）

也可以直接使用命令行脚本：

```bash
# 文生图
./generate_image.sh "你的提示词"

# 图生图编辑
./edit_image.sh "编辑提示词" original.jpg
```

所有图片同样保存到 `images/` 目录，记录到 `generation.log`。

## 项目结构

```
.
├── .env                    # API 密钥（不上传 Git）
├── .gitignore              # Git 忽略配置
├── app.py                  # Flask Web 后端
├── requirements.txt        # Python 依赖
├── templates/
│   └── index.html         # 网页前端
├── generate_image.sh       # 命令行文生图
├── edit_image.sh           # 命令行图生图
├── generation.log          # 生成日志（所有提示词和参数）
├── images/                 # 所有生成的图片保存目录
├── index.html              # 静态展示最新图片（原始静态版）
└── README.md
```

## 配置参数

在 `app.py` 中可以修改：

```python
MODEL = 'doubao-seedream-5-0-260128'  # 模型名称
SIZE = '2K'                              # 生成图片尺寸
PORT = 5000                              # 服务端口
```

## 说明

- 每张图片都用时间戳命名 `generated_YYYYMMDD_HHMMSS.jpg`，不会覆盖历史图片
- 所有生成信息（时间、文件名、模型、尺寸、提示词、耗时）都记录在 `generation.log`
- API Key 永远不会提交到代码仓库，始终保存在本地 `.env` 文件中
